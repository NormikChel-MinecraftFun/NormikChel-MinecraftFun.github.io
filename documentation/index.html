Документация

Подробное описание всех инструментов на сайте.

Калькулятор блоков для дома

Позволяет рассчитать количество блоков, необходимых для строительства дома заданных размеров. Учитывает стены, пол и крышу.

Генератор случайных координат

Генерирует случайные координаты X, Y, Z в допустимых пределах мира Minecraft для телепортации через команду /tp.

Конвертер игрового времени

Преобразует реальные минуты в игровые дни и наоборот. 1 игровой день = 20 минут реального времени.

Калькулятор опыта

Показывает, сколько уровней опыта вы получите за указанное количество очков опыта (XP), согласно формуле Minecraft.

Генератор названий предметов

Создаёт уникальные и креативные названия для предметов, подходящие для модов, карт или ролевых серверов.

Калькулятор фермы криперов

Рассчитывает, сколько криперов нужно убить, чтобы получить желаемое количество пороха (с учётом шанса дропа 100% при обычных условиях).

Генератор случайного seed

Генерирует случайный числовой seed для создания нового мира в Minecraft. Подходит для Java и Bedrock Edition.

Калькулятор прочности брони

Определяет, сколько ударов выдержит броня из разных материалов (кожа, золото, железо, алмаз и т.д.) до полного износа.

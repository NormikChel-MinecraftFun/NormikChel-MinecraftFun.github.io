⛏️ Minecraft 工具中心

你 Minecraft 冒险所需的所有工具

🧱 房屋方块计算器

计算建造房屋所需的方块数量。

🧭 随机坐标生成器

在 Minecraft 世界范围内生成用于传送 (/tp) 的随机 X, Y, Z 坐标。

⏳ 游戏时间转换器

将现实分钟转换为游戏天数，反之亦然。1 天 = 20 分钟。

✨ 经验计算器

显示根据给定的 XP 点数你能获得多少经验等级。

🏷️ 物品名称生成器

生成独特且富有创意的物品名称，非常适合自定义模组或 RPG 服务器。

💥 苦力怕农场计算器

计算在正常条件下（100% 掉落率）获得所需火药数量需要杀死多少只苦力怕。

🌱 随机种子生成器

生成用于创建新 Minecraft 世界的随机数字种子。适用于 Java 版和基岩版。

🛡️ 护甲耐久计算器

确定由不同材料（皮革、金、铁、钻石等）制成的护甲在损坏前能承受多少次攻击。

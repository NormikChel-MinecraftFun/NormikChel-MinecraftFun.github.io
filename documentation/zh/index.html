文档

本网站所有工具的详细说明。

🧱 房屋方块计算器

计算建造指定尺寸房屋所需的方块数量，包括墙壁、地板和屋顶。

🧭 随机坐标生成器

在 Minecraft 世界范围内生成随机 X、Y、Z 坐标，用于 /tp 命令。

⏳ 游戏时间转换器

将现实分钟与游戏内天数相互转换。1 游戏日 = 20 现实分钟。

✨ 经验值计算器

根据 Minecraft 经验公式，显示指定经验值可获得多少等级。

🏷️ 物品名称生成器

为物品生成独特且富有创意的名称，适用于模组、自定义地图或角色扮演服务器。

💥 苦力怕农场计算器

计算需要击杀多少只苦力怕才能获得指定数量的火药（假设 100% 掉落率）。

🌱 随机种子生成器

生成用于创建新 Minecraft 世界的随机数字种子，适用于 Java 版和基岩版。

🛡️ 护甲耐久计算器

计算不同材质（皮革、金、铁、钻石等）的护甲在完全损坏前可承受多少次攻击。

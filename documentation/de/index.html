Dokumentation — Minecraft Tools Hub

Ausführliche Beschreibungen aller auf der Website verfügbaren Tools.

🧱 Blockrechner für Häuser

Berechnet die benötigte Anzahl an Blöcken für den Bau eines Hauses mit vorgegebenen Abmessungen, einschließlich Wänden, Boden und Dach.

🧭 Generator von zufälligen Koordinaten

Erzeugt zufällige X-, Y- und Z-Koordinaten für Teleportation (/tp) innerhalb der Minecraft-Welt.

⏳ Umrechner für Spielzeit

Konvertiert reale Minuten in Spieltage und umgekehrt. 1 Tag = 20 Minuten.

✨ Erfahrungsrechner

Zeigt an, wie viele Erfahrungsstufen Sie für eine bestimmte Anzahl von XP-Punkten erhalten.

🏷️ Gegenstandsnamengenerator

Erstellt einzigartige und kreative Namen für Gegenstände, ideal für benutzerdefinierte Mods oder RPG-Server.

💥 Creeper-Farm-Rechner

Berechnet, wie viele Creepers Sie töten müssen, um eine gewünschte Menge Schießpulver zu erhalten (bei 100 % Drop-Chance unter normalen Bedingungen).

🌱 Zufälliger Seed-Generator

Generiert einen zufälligen numerischen Seed zur Erstellung einer neuen Minecraft-Welt. Funktioniert für Java und Bedrock Edition.

🛡️ Rüstungshaltbarkeits-Rechner

Bestimmt, wie viele Treffer Rüstungen aus verschiedenen Materialien (Leder, Gold, Eisen, Diamant usw.) aushalten, bevor sie kaputtgehen.
